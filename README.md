## Project

**[Decentralized CL Checkpoint Sync](https://github.com/eth-protocol-fellows/cohort-seven/blob/master/projects/project-ideas.md#decentralized-cl-checkpoint-sync)** (Nimbus), mentored by Etan Kissling.

The project replaces the current trust-based checkpoint sync — where new nodes download a state snapshot from a trusted server — with a peer-to-peer approach extending the Altair light client sync protocol, so new nodes can bootstrap trustlessly from peers instead.
